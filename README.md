# メモ帳アプリ

React + Vite で構築したシンプルで多機能なメモ帳アプリケーションです。

## 機能

### 主要機能
- **複数メモ管理**: メモの作成、編集、削除が可能
- **マークダウン対応**: マークダウン記法でメモを書き、リアルタイムでプレビュー
- **タグ機能**: メモにタグを付けて分類・フィルタリング
- **検索機能**: タイトルや内容でメモを素早く検索
- **自動保存**: 編集内容を自動的に保存（1.5秒のデバウンス）

### 表示モード
- **編集モード**: マークダウンエディタのみ表示
- **分割モード**: エディタとプレビューを並べて表示
- **プレビューモード**: プレビューのみ表示

### その他の特徴
- **ローカルストレージ**: ブラウザのlocalStorageにデータを保存
- **XSS対策**: DOMPurifyによるマークダウンHTML出力のサニタイズ
- **レスポンシブデザイン**: デスクトップ、タブレット、モバイルに対応
- **リアルタイム検索**: デバウンス処理による快適な検索体験

## 技術スタック

- **React 18.3** - UIライブラリ
- **Vite 5.4** - ビルドツール＆開発サーバー
- **marked** - マークダウンパーサー
- **DOMPurify** - HTMLサニタイザー（XSS対策）
- **uuid** - ユニークID生成

## セットアップ

### 必要要件
- Node.js 16.0 以上

```bash
# 依存関係のインストール

```


```bash
# 本番用にビルド
npm run build

# ビルド結果をプレビュー
npm run preview
```

## 使い方

### メモの作成
1. ヘッダーの「新規メモ」ボタンをクリック
2. タイトルと内容を入力
3. 自動的に保存されます

### タグの追加
1. メモ編集画面のタグ入力欄にタグ名を入力
2. Enterキーで確定
3. タグをクリックして削除可能

### メモの検索
1. サイドバー上部の検索バーに検索キーワードを入力
2. タイトル、内容、タグから該当するメモが表示されます

### タグでフィルタリング
1. サイドバーの「タグでフィルタ」セクションでタグをクリック
2. 選択したタグを含むメモのみ表示されます

### 表示モードの切り替え
ヘッダーのアイコンをクリックして、編集/分割/プレビューモードを切り替えられます。

### メモの削除
1. 削除したいメモを開く
2. 右上のゴミ箱アイコンをクリック
3. 確認ダイアログで「削除」を選択

## データの保存

アプリケーションは、すべてのメモデータをブラウザのlocalStorageに保存します。

- **保存先**: `localStorage` キー: `memo-app-data`
- **容量制限**: 約5-10MB（ブラウザにより異なる）
- **データ形式**: JSON

### データのバックアップ

localStorageは永続的ではないため、重要なメモは定期的にバックアップすることをお勧めします。ブラウザの開発者ツールからlocalStorageの内容をエクスポートできます。

## マークダウン記法

このアプリはGitHub Flavored Markdown（GFM）に対応しています。

### 使用例

```markdown
# 見出し1
## 見出し2

**太字** *斜体* ~~取り消し線~~

- リスト項目1
- リスト項目2

1. 番号付きリスト1
2. 番号付きリスト2

[リンク](https://example.com)

> 引用文

\`インラインコード\`

\`\`\`
コードブロック
\`\`\`
```

## セキュリティ

- **XSS対策**: DOMPurifyでマークダウンHTMLをサニタイズ
- **データ保護**: localStorageは暗号化されないため、機密情報の保存は避けてください
